Project by: Ukamaka Nwobodo.

Project Date: 23-04-2023.

Project Technologies: Web Application/Bootstrap/Fontawesome.

Project Languages: HTML5, CSS, Javascript..

How To

Click on the game area to start.

Click on the game area / press any key to jump the obstacle.

The more obstacles you jump the more your score.

This project uses dom element manipulation, transition, scale and rotation to describe to a secondary school student the basics of computer graphics animations.
